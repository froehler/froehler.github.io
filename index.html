Übersicht

Die Software rfWärme wird unter der GNU General Public License Version 3 veröffentlicht und kann somit kostenfrei genutzt werden. Als Betriebssystem wird Linux verwendet. Bei der Wahl einer geeigneten Distribution sollte darauf geachtet werden, dass darin die Programmierumgebung Gambas enthalten ist. Empfohlen werden hier Debian oder Ubuntu. Das Programm rfWärme wurde in Gambas erstellt und benötigt diese Umgebung auch für den Betrieb.
